## Install

**Linux x64:**
```bash
curl -Lo bannkenn-agent https://github.com/OpenTech-Lab/bannkenn/releases/download/v1.4.10/bannkenn-agent-linux-x64
curl -Lo bannkenn-containment.bpf.o https://github.com/OpenTech-Lab/bannkenn/releases/download/v1.4.10/bannkenn-containment-linux-x64.bpf.o
sudo install -m 755 bannkenn-agent /usr/local/bin/bannkenn-agent
sudo install -d /usr/lib/bannkenn/ebpf
sudo install -m 644 bannkenn-containment.bpf.o /usr/lib/bannkenn/ebpf/bannkenn-containment.bpf.o
```

**Linux ARM64:**
```bash
curl -Lo bannkenn-agent https://github.com/OpenTech-Lab/bannkenn/releases/download/v1.4.10/bannkenn-agent-linux-arm64

```
